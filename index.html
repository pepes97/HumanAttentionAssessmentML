Welcome to our Visual Perception in HCI & HRI Project

This project have been developed for the Master's Course "Elective in Artificial Intelligence 2021" of Professor C. Napoli, at Sapienza University of Rome. The aim of the project is to create a machine learning system to assess the interlocutor attention in a human-robot or human-computer interaction.

About the project

Contributions

Many human-computer or human-robot interactions require the capability of the system of understanting whether the user is paying attention or not. However, to train such systems, large amounts of data are needed, but they are currently unavailable. In this paper, we focus on the attention detection task in its entirety, from addressing the issue of data paucity, passing through the creation of a baseline system and the proposal of novel strategies to further improve performances, up to the development of this web application. More specifically, the main contributions of our project can be summarized as follows:

  • We introduce a new manually-annotated dataset for the attention detection the task, containing about 120k images from 18 different users;
  • We develop a strong baseline system for the task, which is able to achieve competitive results on the proposed dataset;
  • We enhance our baseline system by adding an auxiliary face detection module, achieving a boost in performances;

  • We propose a novel GAN-based data augmentation technique to further enrich the collected data, again attaining performance improvements;
  • We extensively evaluate the benefits of our contributions on the proposed test set by performing a statistical analysis on the obtained results;
  • We design a web application to further analyze the behavior of the system and enable real-time testing.

Applications

what this system could be useful for


Children affected by ADHD

When dealing with children affected by hyperactivity disorder, understanding whether the child is following or not the proposed activity (e.g., an educational lesson) is needed, in the latter case, to propose again the same activity, or pause it.


Standard HRI interaction

In a standard HRI interaction, if the user is not heedful when the robot asks a question (e.g., "What do you want to order?", in a restaurant scenario), it can turn up the volume and repeat the question to grab attention.


Driver monitoring system

In a driver monitoring system, check whether the driver is not looking ahead can be useful to alert him, and bring his attention back.
